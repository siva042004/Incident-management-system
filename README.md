# Incident Management System

## 📌 Project Overview
A full-stack Incident Management System built using Spring Boot, React, and MySQL.

## 🚀 Tech Stack
- Backend: Spring Boot, JPA, Hibernate
- Frontend: React.js
- Database: MySQL
- Build Tool: Maven
- Version Control: Git & GitHub

## 📂 Project Structure

## ⚙️ How to Run

### Backend
cd backend
mvn spring-boot:run

Runs on: http://localhost:8080

### Frontend
cd frontend
npm install
npm start

Runs on: http://localhost:3000

## ✨ Features
- Create Incident
- View Incidents
- REST API Integration
- Database Persistence
